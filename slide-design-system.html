Why it works

Five principles we return to.

Decisions that have held up across projects, budgets, and clients over the past six years.

  • Clarity over cleverness. The first read is the only read that matters.
  • Typography is 80% of the work. Get it wrong and nothing else recovers.
  • Color is a decision, not a default. Commit or leave it out.
  • Motion should reward, not announce. Subtle beats showy every time.
  • Every surface is an opportunity to reinforce the voice.

Layouts

Each slide type is a single-file component in components/layouts. Add your own by dropping in a new .vue file.

Theme

All theme logic lives in useTheme — one composable resolves the active theme once per page.

Content

Slide data comes from Storyblok via useSlides. The schema mirrors the slide component props 1:1.
